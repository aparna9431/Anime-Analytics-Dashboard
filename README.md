# Anime-Analytics-Dashboard
Designed and developed an interactive Power BI dashboard to analyze trends and audience engagement across 600+ anime titles. The dashboard provides data-driven insights into anime production, popularity, and genre distribution through dynamic and visually engaging analytics.

Built multiple analytical visuals to uncover insights, including top producing studios, most popular genres, anime distribution by type, top-rated anime shows, and airing status trends. Applied data cleaning, transformation, and DAX calculations to ensure accurate and meaningful analysis.


Tools & Skills: Power BI, Power Query, DAX, Data Visualization, Data Cleaning, Dashboard Design

Key Features

Genre Analysis: Identifies the most popular and highest-rated genres over time. 
Studio Performance: Tracks studio production volume and audience reception metrics. 
Yearly Trends: Visualizes the growth of the anime industry by release year and season. 
User Ratings & Popularity: Correlation analysis between user scores and overall popularity. 
Dynamic Filtering: Interactive slicers for genres, years, and production studios to allow for customized views.

Dashboard Insights

Top Studios: Comparison of studios based on the number of high-rated titles produced.
Genre Shifts: Analysis of how viewer preferences have shifted from traditional genres to modern trends.
The "Seasonal Effect" on Engagement : Using the release season data (Spring, Summer, Fall, Winter), to analyze if timing impacts an anime's success.
Correlation: Popularity vs. Rating : This insight explores the relationship between how many people watched an anime versus how they actually rated it.
